=====================================================
  BUG FIXES
=====================================================

1. ✅ BUG: Speed x3 stuck / animasi beku saat pertandingan
   - Root cause: ball.celebration jalan 3x per render frame di dalam loop speed
     menyebabkan bola keluar lapangan & engine freeze
   - Fix: celebration movement dipindah ke render phase (1x per frame)
   - Fix: Semua setTimeout di timeline events dihapus (tidak aman di speed tinggi)
   - Dibuat _buildFixedTimeline() baru yang 100% synchronous

2. ✅ BUG: Formasi tidak bisa ganti player dari bench
   - Root cause: HTML5 drag-and-drop tidak bekerja di touch/mobile
   - Fix FRONTEND: Tambah click-to-select mechanic
     → Klik bench player → muncul hint "Pilih posisi di lapangan"
     → Klik posisi di lapangan → player langsung ditempatkan
   - Fix BACKEND: setFormationPlayer lebih fleksibel
     → GK tetap ketat (GK only untuk GK)
     → Non-GK bisa cross-position swap (misal CM ke ST)
     → Auto-swap jika posisi sudah terisi (player lama → posisi lama player baru)
   - Fix MOBILE: Touch drag-and-drop polyfill untuk layar sentuh
